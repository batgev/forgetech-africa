THIS IS A FULL STACK TODO APPLICATION!!

IT WILL ACHIEVE THE FOLLOWING;
1. SECURE LOGIN
2. CREATING A TODO
3. COMPLETING A TODO
4. SET EXPIRATION FOR THE COMPLETION OF A TODO
4. UPDATE A TODO
5. DELETE A TODO

HOW IT WILL WORK!
*ALL THE DATA OF THE USER(USERNAME,PASSWORD,TODO's) WILL BE STORED IN A POSTGRES DATABSE.
*SIGNUP & LOGIN REQUEST WILL BE SENT TO THE BACKEND FOR REGISTRATION AND AUTHENTICATION.
*USERS WILL BE ASSIGNED A TOKEN UPON LOGING IN.
*USERS CAN NOW PERFORM ALL THE ACTIONS OF CREATING,COMPLETING,UPDATING AND DELETING A TODO
